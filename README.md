# Task Tracker

A simple task tracking app built with React Native, Expo, and TypeScript.

## Setup

Install dependencies

npm install

Start the project

npx expo start

## Features

- Create tasks
- Mark tasks complete or incomplete
- Filter tasks (All / Active / Completed)
- Tasks persist using AsyncStorage
- Input validation prevents empty tasks

## Libraries Used

@react-native-async-storage/async-storage

Used to persist tasks locally on the device.

uuid

Used to generate unique IDs for tasks.
